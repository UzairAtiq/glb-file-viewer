- `image1.jpg` - First image for left sidebar
- `image2.jpg` - Second image for left sidebar  
- `image3.jpg` - Third image for left sidebar
- `image4.jpg` - First image for right sidebar
- `image5.jpg` - Second image for right sidebar

**Note:** The images you provided in the attachments should be saved with these names. If the images don't load, placeholder images will be shown instead.
